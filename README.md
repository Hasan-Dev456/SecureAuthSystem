# Secure Authentication System

A secure JavaFX authentication system connected to MySQL with password hashing and login attempt tracking.

---

## Features

- User Registration
- Secure Login System
- SHA-256 Password Hashing
- Failed Login Attempt Tracking
- JavaFX User Interface
- MySQL Database Integration
- Modern CSS Styling

---

## Technologies Used

- Java
- JavaFX
- MySQL
- JDBC
- CSS
